# Show da Heloísa

Aplicação web interativa no formato de jogo de perguntas e respostas, desenvolvida com foco em experiência do usuário, engajamento e lógica de aplicação no front-end.

---

## Sobre o projeto

O Show da Heloísa é um jogo online acessível pelo navegador que combina elementos de quiz com mecânicas de jogo, como sistema de vidas, tempo, dicas e ranking de jogadores.

O projeto foi desenvolvido com o objetivo de praticar conceitos reais de desenvolvimento web, incluindo autenticação, persistência de dados e interação dinâmica com o usuário.

---

## Funcionalidades

- Sistema completo de perguntas e respostas
- Controle de tempo por rodada
- Sistema de vidas
- Sistema de dicas durante o jogo
- Feedback imediato de acertos e erros
- Ranking global de jogadores
- Autenticação de usuários via Firebase
- Persistência de dados em tempo real
- Interface interativa com suporte a efeitos sonoros
- Experiência responsiva para diferentes dispositivos

---

## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript
- Firebase (autenticação e banco de dados)
- Deploy na Vercel
